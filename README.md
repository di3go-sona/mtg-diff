# mtg-diff

Compare two MTG decklists and see exactly what changed.

```
npm install
npm run dev    # dev with live reload
npm start      # production
```

Open `http://localhost:3000`.

## usage

1. Paste your original deck in **source**, new deck in **target**
2. Hit `>_ compare decks`
3. See what to cut and what to buy
4. Click **print proxies** to open a print-ready page with Scryfall images

## format

Paste standard Arena / MTGO format:

```
4 Lightning Bolt
3 Ragavan, Nimble Pilferer

Sideboard:
2 Flusterstorm
```

Hover any card name to preview the card art.
